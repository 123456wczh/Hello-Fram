# 全局配置文档 (Documentation for config.py)

**文件位置**: `src/config.py`
**功能**: 存储全局常量、游戏平衡参数及 UI 布局设置。

## 📜 代码逐行解析

| 行号 | 代码内容 | 解析与说明 |
| :--- | :--- | :--- |
| 4-5 | `SCREEN_WIDTH`, `HEIGHT` | 设定窗口分辨率为 1280x720 (16:9)。这是现代“宽屏 OS”风格的基础。 |
| 8 | `GAME_AREA_WIDTH` | 游戏渲染区域宽度。当前设为全屏宽度，编辑器会覆盖在上面。 |
| 9 | `UI_PANEL_WIDTH = 500` | 代码编辑器侧边栏滑出时的宽度。500px 足够容纳宽行代码。 |
| 10 | `GRID_SIZE = 60` | 每个农场地块的像素大小。 |
| 13-17 | `COLOR_...` | 全局配色表。修改这里可以一键更换游戏的主题色调。 |
| 20 | `SAVE_FILE` | 存档文件名为 `savegame.json`，位于根目录。 |
| 23 | `DRONE_MOVE_DELAY = 0.2` | **平衡参数**：无人机移动一步需要 0.2 秒。减小此值会让游戏节奏变快。 |
| 25-26 | `GRID_WIDTH`, `HEIGHT` | 农场大小定义为 10x10 格。 |

## 🛠️ 维护与扩展指南

### 如何调整游戏难度？
*   **修改 `GRID_WIDTH / HEIGHT`**: 可以创建更大的地图 (例如 20x20)。注意性能消耗（主要是渲染循环）。
*   **修改 `DRONE_MOVE_DELAY`**: 设为 `0.1` 可以让脚本执行速度翻倍。

### 如何更换主题？
*   修改 `COLOR_BG` 和 `COLOR_GAME_BG` 即可改变背景氛围。
*   注意 `COLOR_GRID` 的对比度，确保网格线清晰可见。
