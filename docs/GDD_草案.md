# GDD: Hello Farm - 重定义编程仿真生态 (Draft v1.0)

## 1. 游戏核心愿景 (Core Vision)
构建一个基于代码控制的“微缩生态循环”。玩家作为一名“数字农夫”，通过 Python 脚本（或游戏内封装 API）驱动无人机，将一片荒地转化为高度自动化的赛博农场。

## 2. 核心循环 (Core Loop)
1. **编程/配置**：用户在 IDE 窗口输入逻辑。
2. **执行/观察**：无人机在 2D 农场网格中执行任务，作物在可视化环境中生长。
3. **收益/解析**：收割作物获得“信用点（Credits）”与“经验值（XP）”。
4. **进化/解锁**：在**技能树**中解锁更高级的传感器、API 或种子，回到步骤 1 优化代码。

## 3. 技能树系统 (The API Tech Tree)
| 阶段 | 解锁功能 | 玩家挑战 |
| :--- | :--- | :--- |
| **Lvl 1: 种子用户** | `move()`, `plant(carrot)`, `harvest()` | 基础 `while` 循环与移动步长计算。 |
| **Lvl 2: 逻辑感知** | `get_pos()`, `inspect()` | `if` 条件判断，根据作物成熟度决定收割。 |
| **Lvl 3: 空间规划** | `coordinate_system`, `map_size` | 编写双层循环（嵌套循环）覆盖全图。 |
| **Lvl 4: 异形生态** | `Sunflower.sort()`, `Cactus.safe_zone()` | 算法挑战：排序、空间避障、优先级队列。 |

## 4. 创意作物逻辑 (Creative Crops)
- **胡萝卜 (Carrot)**：基础作物。成长周期恒定，最适合入门练习。
- **南瓜 (Pumpkin)**：拼图作物。需要 4 个成熟南瓜在 2x2 网格内相邻，可“融合”为高级巨大南瓜（奖励翻倍）。
- **向日葵 (Sunflower)**：动态目标。它会存储一个随机值，无人机收割时的逻辑必须按从小到大的顺序，否则收益折损。

## 5. 交互与视觉设计 (Visual & Interaction)
- **UI 风格**：赛博朋克极简风。编辑器具有语法高亮，控制台输出具有日志级别颜色。
- **反馈效果**：无人机移动时带有微小的拖尾，收割时有金币跳出特效。
- **引导系统**：任务面板位于右上角，每一步操作伴随“闪烁提示”，引导玩家从第一行 `while True` 开始。
